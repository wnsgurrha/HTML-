HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. web browsers recive HTML documents form a web server of from local storage and render them into multimedia web pages.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as internet forms, maybe embadded into the rendered page.
